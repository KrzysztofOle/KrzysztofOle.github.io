# Krzysztof Olejnik — personal website

Bilingual personal website presenting experience in industrial robotics, machine vision, software development, AI and accounting automation.

- Polish: <https://krzysztofole.github.io/>
- English: <https://krzysztofole.github.io/en/>
- Hosting: GitHub Pages
- Deployment: GitHub Actions from `main`

The site is built with semantic HTML, CSS and a small amount of vanilla JavaScript. It has no runtime dependencies.
